# Очистка устаревших данных в системе слежения

**Название приложения:** cc3cleaner.

**Назначение приложения:** очистка устаревших данных в системе слежения за регистрируемыми почтовыми отправлениями в отделении почтовой связи.

Приложение используется инженерами группы технической поддержки для обслуживания системы слежения за регистрируемыми почтовыми отправлениями.


Приложение написано на **Python 2.7** с использованием библиотеки кроссплатформенного графического интерфейса пользователя (GUI) **WxPython**.


Используются модули, не входящие в стандартную поставку интерпретатора:
        - wxPython.
